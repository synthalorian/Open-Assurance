# Sound Assets

Place ambient sound files here:

- `rain.wav` - Gentle rainfall
- `waves.wav` - Ocean waves
- `forest.wav` - Forest ambiance
- `wind.wav` - Soft wind

## Recommended Format
- Format: WAV or MP3
- Sample Rate: 44100 Hz
- Bit Depth: 16-bit
- Channels: Mono or Stereo
- Duration: 30-60 seconds (will loop)
- Size: Keep under 2MB per file

## Free Sound Sources
- [Freesound](https://freesound.org) (CC0 license)
- [Pixabay](https://pixabay.com/music/) (Free for commercial use)
- [BBC Sound Effects](https://sound-effects.bbcrewind.co.uk/) (Personal use)

## Note
The app will still work without these files - it will simply skip audio playback.
